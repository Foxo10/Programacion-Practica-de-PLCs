The file "GDIPLUS.DLL" provided with the software is a library created by Microsoft.
This library is part of the operating system from Windows XP.
If you use the software on Windows XP or Vista, you can delete this file.
If you use the software on an earlier version of Windows, it is possible that the library is not present ; that's why it comes with the software.
If you use the software from an USB key, it is better to have this file in the same directory as the executable file of the software, in case of if you use the software on a computer which doesn't have this library.
